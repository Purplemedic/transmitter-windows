# transmitter-windows
Python-based data acquisition tool for Mettler Toledo transmitters (Conductivity, pH, etc.) via USB/HID on Windows. Features real-time sensor monitoring and data logging.
##  Key Features
* **Multi-Parameter Support:** Interface with sensors for pH, Conductivity, temperature,and more.
* **Protocol Support:** Optimized for HID/USB communication (compatible with Mettler Toledo M200/M300/M800 series).
* **Data Logging:** Automated data extraction for process monitoring.

## 💻 Prerequisites
* **OS:** Windows 10/11
* **Driver:** Ensure the correct USB/HID or Serial drivers are installed for your transmitter.
* **Python 3.x**
* **Mettler Toledo Transmitter M200/M300/M800
